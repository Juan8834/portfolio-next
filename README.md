## 📦 Installation & Setup

### Clone the repository
```bash
git clone https://github.com/Juan8834/Juan8834.github.io.git
cd portfolio-next
Install dependencies
npm install
Run the development server
npm run dev

Open in browser: http://localhost:3000

📁 Project Structure
components/
  Navbar.js
  Footer.js
  ProjectCard.js

pages/
  index.js
  projects.js
  about.js
  contact.js

public/images/
styles/
```
